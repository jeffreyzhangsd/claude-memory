description: All wiki pages should be written in plain language for a first-time ML learner, not academic style

Write wiki topic pages in plain, accessible language — "explain like a smart young adult who hasn't taken college math." Do not assume knowledge beyond high school (basic algebra, some geometry, no calculus, no linear algebra, no probability theory). Use analogies and concrete examples. Avoid dense notation-heavy paragraphs. Still include the math and formulas, but always explain them in plain English first.

**Why:** User said this explicitly when starting MML book ingest — "it's my first time learning about machine learning really, explain like I'm a young adult."

**How to apply:** Every new wiki page and every update to existing CS229 pages should follow this tone. When updating CS229 pages, rewrite dense sections to be more conversational and concrete.
